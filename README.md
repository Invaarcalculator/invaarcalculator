## Over Invaarcalculator

Invaarcalculator is een rekentool voor SSPF-gepensioneerden na de AOW-leeftijd. De tool maakt inzichtelijk hoe het “invaren” naar de nieuwe pensioenregeling kan doorwerken in het persoonlijke aandeel of pensioenkapitaal, afhankelijk van onder meer de dekkingsgraad en de verdeling van de buffer. De calculator is bedoeld als transparante scenariohulp: gebruikers kunnen aannames en effecten verkennen, maar de uitkomst is geen officiële pensioenopgave.

## Privacy

Invaarcalculator draait volledig lokaal in de browser. Ingevoerde gegevens, zoals leeftijd, jaarpensioen, partnergegevens en aangepaste parameters, worden niet naar een server verzonden, niet opgeslagen en niet gedeeld met derden. Alle berekeningen vinden plaats in JavaScript in de browser; na het sluiten van het tabblad of de browser zijn de ingevoerde gegevens weg.

De tool gebruikt geen cookies, geen webanalyse en geen tracking-scripts.

De website wordt gehost via GitHub Pages. GitHub kan bij het laden van de pagina standaard technische servergegevens verwerken, zoals IP-adres, browserinformatie, tijdstip van bezoek en opgevraagde bestanden, voor beveiligings- en infrastructuurdoeleinden. Dit valt buiten de invloed van de auteur.
